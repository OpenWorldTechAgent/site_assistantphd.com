# Tech Stack: assistantphd.com

## Frontend
- **Framework:** Next.js (React) - For a high-performance, SEO-friendly modern web application.
- **Styling:** Tailwind CSS - Utility-first CSS framework for rapid UI development and a clean, minimalist aesthetic.

## Backend
- **Framework:** Next.js API Routes (Node.js) - Integrated serverless functions for handling application logic and API requests.

## Database
- **Primary Database:** PostgreSQL - Reliable relational database for storing structured data like service details and client inquiries.

## Infrastructure & Tools
- **Version Control:** Git
- **Package Manager:** npm or yarn
